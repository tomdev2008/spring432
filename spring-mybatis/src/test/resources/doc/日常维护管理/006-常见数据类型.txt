##日期和时间

------------          	---------   	---------------------                      	----------------------------------------- 
DATETIME      		 8 bytes   	YYYY-MM-DD HH:MM:SS    				1000-01-01 00:00:00 ~ 9999-12-31 23:59:59 
DATE           		 3 bytes   	YYYY-MM-DD                      		1000-01-01                ~ 9999-12-31 
TIME           		 3 bytes  	HH:MM:SS              				-838:59:59          ~ 838:59:59
